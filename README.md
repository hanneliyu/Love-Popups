### 1. Program: Love Popups 
**程序介绍：情感弹窗**  
Love is like a virus, cause I can't stop thinking about you. I think I'm infected.  
恋爱就像病毒。我发现对你上了瘾，无法无法停止想念你。  
  
    
  
### 2. Key Features  
**核心特征：**  
- Auto-popups: The program keeps generating windows (up to 300) once launched, with random positions on the screen.  
  自动弹窗：不断生成最多 300 个窗口，随机分布在屏幕上  

- Multilingual support: Covers English, Chinese, Spanish, and emojis, adapting to different language environments.  
  多语言支持：涵盖英语、中文、西班牙语及表情符号  

- Safe exit mechanisms:  
  安全退出机制：  
  - Auto-exits after 15 seconds to avoid constant disruption.  
    15 秒后自动退出，避免持续干扰  
  - Supports shortcut key closure (Ctrl+Alt+Q or Ctrl+C).  
    支持快捷键关闭（Ctrl+Alt+Q 或 Ctrl+C）  
  - A hidden main window has an exit button (uncomment root.deiconify() to display it).  
    隐藏的主窗口中设有退出按钮（取消注释 root.deiconify() 即可显示）
